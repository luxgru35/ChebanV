## Описание проекта

REST API для управления мероприятиями с поддержкой:
- CRUD операций для мероприятий и пользователей
- Soft delete (мягкое удаление)
- CORS-ограничений с настройкой доверенных доменов
- Swagger документации
- Логирования запросов

## Технологии

- Node.js
- Express.js
- PostgreSQL
- Sequelize ORM
- Swagger (документация API)
- Morgan (логирование)

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Создайте базу данных PostgreSQL:
```sql
CREATE DATABASE events_db;
```

3. Настройте файл `.env` (см. `.env.example`)

4. Запустите сервер:
```bash
npm run dev
```

## API Документация

После запуска сервера документация доступна по адресу:
http://localhost:5000/api-docs

## Endpoints

### Users
- `GET /users` - Получить список пользователей
- `POST /users` - Создать пользователя
- `DELETE /users/:id` - Удалить пользователя (soft delete)

### Events
- `GET /events` - Получить список мероприятий
- `GET /events/:id` - Получить мероприятие по ID
- `POST /events` - Создать мероприятие
- `PUT /events/:id` - Обновить мероприятие
- `DELETE /events/:id` - Удалить мероприятие (soft delete)
